This package provides HubSpot Http support for [HubSpot's API](https://developers.hubspot.com/docs/overview).

## Installation

To install, use composer:

```
composer require flipbox/http-hubspot
```

## Testing

``` bash
$ ./vendor/bin/phpunit
```
